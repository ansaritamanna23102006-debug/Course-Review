1. PROJECT OVERVIEW
================================================================================

CourseReview is a modern, full-stack course review platform built with Next.js 
14, React 18, MongoDB, and Tailwind CSS. The application enables users to share, 
discover, and manage online course reviews across major learning platforms.

Project Name: course-review-system
Description: A full-stack course review application built with Next.js

API ENDPOINT 1: GET /api/reviews
────────────────────────────────────────────────────────────────────────────

Description: Fetch all reviews with pagination, filtering, and sorting

HTTP Method: GET

Query Parameters:
  - page (number, optional, default: 1)
    - Page number for pagination
  
  - limit (number, optional, default: 10)
    - Reviews per page
  
  - rating (number, optional)
    - Filter by rating (1-5)
  
  - search (string, optional)
    - Search by course title (case-insensitive)
  
  - sort (string, optional, default: latest)
    - Sort order: latest, oldest, highest, lowest

Example Request:
  GET /api/reviews?page=1&limit=10&sort=highest&rating=5&search=react

Success Response (HTTP 200):

API ENDPOINT 2: POST /api/reviews
────────────────────────────────────────────────────────────────────────────

Description: Create a new course review

HTTP Method: POST

Request Body:
{
  "courseTitle": "React from Scratch",
  "platform": "Udemy",
  "name": "Alice Johnson",
  "rating": 5,
  "description": "This course is fantastic! I learned so much about React hooks..."
}

Request Body Schema:
  - courseTitle (string, required)
    - Length: 1-200 characters
  
  - platform (string, required)
    - Enum: Udemy, Coursera, YouTube, LinkedIn Learning, Pluralsight, Other
  
  - name (string, optional)
    - Length: 0-100 characters
    - Defaults to "Anonymous" if not provided
  
  - rating (integer, required)
    - Allowed values: 1, 2, 3, 4, 5
  
  - description (string, required)
    - Length: 10-1000 characters

Success Response (HTTP 201):
{
  "success": true,
  "message": "Review created successfully",
  "data": {
    "_id": "507f1f77bcf86cd799439013",
    "courseTitle": "React from Scratch",
    "platform": "Udemy",
    "name": "Alice Johnson",
    "rating": 5,
    "description": "This course is fantastic!...",
    "createdAt": "2024-01-15T12:00:00.000Z",
    "updatedAt": "2024-01-15T12:00:00.000Z"
  }
}

Error Response - Missing Fields (HTTP 400):
{
  "success": false,
  "error": "Missing required fields: courseTitle, platform, rating, description"
}

Error Response - Validation Error (HTTP 400):
{
  "success": false,
  "error": "Validation failed",
  "details": [
    "Review must be at least 10 characters",
    "Rating must be at least 1"
  ]
}

Error Response - Server Error (HTTP 500):
{
  "success": false,
  "error": "Failed to create review"
}

API ENDPOINT 3: DELETE /api/reviews/:id
────────────────────────────────────────────────────────────────────────────

Description: Delete a specific review (admin only, password-protected)

HTTP Method: DELETE

Path Parameters:
  - id (string, required)
    - MongoDB review ID (24-character hexadecimal string)

Request Body:
{
  "adminPassword": "admin123secure"
}

Example Request:
  DELETE /api/reviews/507f1f77bcf86cd799439011

Success Response (HTTP 200):
{
  "success": true,
  "message": "Review deleted successfully",
  "data": {

  }
